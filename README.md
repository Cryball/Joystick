## :rocket: **Installation**

Choose server folder by:

```shell
cd server
```

And run: 

```shell
npm start
```

___

Then open cecond terminal, choose client folder:

```shell
cd client
```

Install all dependencies by:

```shell
npm install
```

Start web app by: 
```shell
npm start
```
